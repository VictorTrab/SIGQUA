# Reglas operativas de Codex para SIGQUA

## Proposito
Este directorio organiza las guias de trabajo de Codex separadas por `roles/` y `skills/`.

Este directorio contiene reglas operativas para Codex. La documentacion del proyecto no se guarda aqui ni en carpetas internas duplicadas; se gestiona en la boveda externa solo cuando sea necesaria.

## Como usar esta carpeta
- `agents.md` sigue siendo la regla global.
- Cargar 1 rol principal por tarea.
- Cargar 0 o 1 skill de apoyo si la tarea realmente lo necesita.
- Evitar abrir varios roles a la vez salvo que una tarea cambie claramente de naturaleza.

## Estructura
- `roles/`: reglas por responsabilidad principal.
- `skills/`: flujos de trabajo breves para ejecutar tareas repetibles.
